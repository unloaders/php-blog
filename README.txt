Inscription , connexion : les champs ne sont pas vérifié mais je pense pas que cela était demandé.
seul les personnes connectés peuvent modifier, supprimer et rédiger un article
Pour la recherche je pense qu'elle est completement fonctionnel avec plusieurs mot clés.
Lors de l'upload d'une image, une nouvelle image redimensionné est stocké et le nom de cette image est 
stocké dans la BD pour l'affichage dans l'acceuil. ### Attention ### ne pas upload une image autre qu'au format jpg, ou une image extrement grande.
Mon cookie avec le SID est bien généré et la BD est bien mis à jours avec celui-ci.
